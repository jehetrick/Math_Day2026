### MCpi

Included above is the raw HTML file for the site:  https://jhetrick.io/MCpi

I *vibe coded* this page using https://perplexity.ai, with the prompt:

*I am giving a presentation on data science and monte carlo methods.
I want to point the participants (about 200) to a website where they can click a button to throw darts. When they click the button, the website will "throw 10 random darts" at a dart board. The dart board is a square of size 2x2. The web site needs to record 10 random x,y coordinates between -1 and 1 for both x and y in a file, updating the file with each participant's click. The webpage should display 10 additional points the 2x2 square, with a circle of radius 1 inside the square. Every time someone clicks the button on the webpage, 10 new points appear in the square and are recorded in the file. The webpage should also keep a running count of the fraction of points that lie inside the circle (i.e. have radius <= 1). Please provide an html page that implements this.*

Follow-up prompt:

*Add a "Clear" button to return the webpage to its inital 0 state, and a dropdown to allow the user to throw either 1, 10, 100, 1000, or 10000 darts.*
